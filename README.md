## ⚡ Overview
**LEONUKEG v2** is a professional, high-performance portfolio designed for full-stack developers. It features a brutalist, technical aesthetic focused on clarity, speed, and atmospheric immersion. Built with a modular architecture, it separates data from presentation to ensure maximum scalability.

## 🛠 Tech Stack
- **Engine:** Vanilla JavaScript (ES6+ Modules)
- **Visuals:** GSAP (ScrollTrigger) for fluid motion
- **Rendering:** Lichtenberg Lightning Engine (Worker-based stochastic generation)
- **Architecture:** Atomic Design System & Dynamic Component Injection
- **Styling:** CSS3 (Variable-driven Design Tokens)

## 🏗 System Architecture
The project follows a strict **Separation of Concerns (SoC)**:
- **`data.js`**: Centralized project and experiment metadata.
- **`renderer.js`**: Atomic component templates for consistent UI rendering.
- **`main.js`**: Orchestrates state management, HUD synchronization, and animations.
- **`lightning.js`**: Offloads heavy procedural calculations to a Web Worker for 60fps performance.
